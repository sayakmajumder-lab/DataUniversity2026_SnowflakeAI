This github repository is created to facilitate the Data University 2026 Snowflake Hands On Session.
The primary focus is on Snowflake AI.
As part of hands on, the participants will perform following tasks: 
- Perform the tasks to setup the environment.
- Create a Cortex Analyst service based on a semantic model for Structured data analysis.
- Create a Cortex Search service based on conversation data for unstructured data analysis.
- Create a Cortex Search service based on Snowflake Product Documentation.
- Creation of custom tools using UDF is not included in this session.
- Create a Cortex Agent and combine all services created beforehand.
- Explore the services and agents using Playground and Snowflake AI.
- Build a Streamlit app (SIS) to expose and explore the Cortex Analyst service using REST API calls.
